# Muhammad Irham Rizq Portfolio

Personal portfolio website for Muhammad Irham Rizq, built with Astro and
styled around the Kuronashiro Hobby visual identity.

## Live Website

https://muhammadirhamrizq-portoweb.vercel.app

## Tech Stack

- Astro
- HTML, CSS, and JavaScript
- Vercel for deployment
- GitHub for source control and CI/CD integration

## Highlights

- CV-based portfolio content
- Featured SalesOps AI case study
- Technical project cards
- GitHub-style tech stack section
- Dark and light mode
- Responsive layout for desktop and mobile

## Commands

```bash
npm install
npm run dev
npm run build
```

## Deployment

The site is deployed on Vercel from the `main` branch of this repository.
Vercel runs `npm run build` and serves the generated `dist/` output.

The supplied assets live in `public/assets/`, and the CV is available from `public/files/cv-muhammad-irham-rizq-2025.pdf`.
